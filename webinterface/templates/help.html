FAQs:

The Monte Carlo simulations require the dicom files for the planning CT image, the plan, the structure set and the individual field doses. These should be exported from Eclipse using the script found under... which will automatically send them to our compute cluster.

When the form is submitted correctly you will be redirected to a page summarizing the details of your simulation. If you are unsure, simply try again as the cluster will not permit the same simulation to be submitted multiple times.

You can only submit a simulation for which the correct dicom files have been exported. If you've done this for the wrong plan, please contact a physicist so that the simulation can be cancelled.

You will not be able to resubmit the job until a physicist has cancelled the one currently running.
